# FullStack Chatroom Room

![](chat-room-demo.gif)



###Libraries & Frameworks
- Express + React + Node.js stack
- Firebase chatroom & user storage
- Socket.io for front-end and server socket managment

## Features
- Cretating a room
- Joining a room
- Blocking a user
- Removing a message
- Ending the room for all users as an admin
- Leaving a room
- Emoji reactions
